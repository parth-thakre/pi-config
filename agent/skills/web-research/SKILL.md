# Web Research

Use only when web/current info is needed.

1. Get candidate URLs with `exa_search` and/or `web_search`.
2. Pick 1–3 best sources.
3. Scrape those URLs with `scrape_url`.
4. Answer from scraped content; cite URLs when factual.

Prefer official/primary/recent sources. If Exa has no key, just use `web_search`.
